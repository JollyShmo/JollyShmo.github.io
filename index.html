ModDB Tools

Carnivores 2

This toolkit includes three tools to simplify 3D modeling and graphics tasks:

  • OBJ_VN_FIXXER: Improves OBJ files by fixing lighting and shading.
  • MTL File Generator: Quickly creates material files for OBJ models.
  • TGA to PNG Converter: Converts TGA images to PNG in batch.

Key Features

These tools make model and texture work faster and easier for artists and developers.

Thief: The Dark Project

The Jolly Thief Toolkit includes two applications:

  • Jolly Thief Mesh Parser v7: Converts Thief object files into OBJ format for easy 3D rendering.
  • Jolly Thief MTL Maker v7: Parses material lists from Thief files and generates MTL files for better material management.

Key Features

These tools allow for object file parsing, easy conversion to OBJ, and user-friendly material management.

Note

Important: Some files may not finish processing, resulting in incomplete models in the game.
